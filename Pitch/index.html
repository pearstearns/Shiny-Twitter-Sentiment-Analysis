Ear to the Ground

Sentiment Analysis with Twitter using Shiny

Created by Ian Roberts

Overview

This tool allows you to interactively perform and visualize sentiment analysis of twitter search terms.

For those not aware, sentiment analysis is a form of natural language processing where the feelings expressed in text data are estimated.

A Look Behind the Curtain

  • the twitteR package is used to interact with twitter and collect the tweets
  • emoticons are stripped
  • the data is passed to the syuzhet (the Russian word for 'the way the story is organized') for the analysis

  • Finally, ggplot and plotly is used to visualize the results

  • The data can be weighted by favorites, done by multiplying the favoriteCount column in the twitter data by the sentiment score.
